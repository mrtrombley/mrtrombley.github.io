The Great Outdoors Near Clemson, SC

More fun things to explore outside.

About This List

This list was started out of a lack of any central place to see all the things that you can do while you're a Clemson student. I hope this can be a resource for students for years to come. If you have any questions/comments/etc. shoot me an email at mrtromb@g.clemson.edu. Enjoy!

Side note: I help run a daily sports newsletter called Sideline Sprint. If you're interested, check it out!
